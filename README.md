# 🔐 Password Manager — Менеджер паролей

Учебный проект для курсовой работы:
«Исследование алгоритмов генерации криптостойких последовательностей символов»

---

## Структура проекта

```
password_manager/
├── main.py          # Точка входа
├── crypto.py        # Криптографические функции (KDF, Fernet)
├── generator.py     # Генерация паролей и парольных фраз
├── storage.py       # Зашифрованное хранилище (JSON)
├── auth.py          # Аутентификация + защита от брутфорса
├── cli.py           # Консольный интерфейс
├── logger.py        # Журнал активности
├── requirements.txt # Зависимости
└── THEORY.md        # Теоретическая часть
```

## Установка и запуск

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

Зависимости:
- `cryptography` — Fernet, PBKDF2, scrypt
- `pyperclip` — копирование паролей в буфер (опционально)

### 2. Запуск

```bash
# Основной режим (хранилище в ~/.password_manager/vault.json)
python main.py

# Произвольный путь к хранилищу
python main.py --vault /path/to/my_vault.json

# Демо-режим (без хранилища, только генерация)
python main.py --demo
```

## Архитектура безопасности

```
Мастер-пароль (в памяти, не сохраняется)
        │
        ├─ PBKDF2-HMAC-SHA256 (600 000 итераций)
        │       + случайная соль (32 байта)
        │       │
        │       ├─► Ключ Fernet (AES-128-CBC + HMAC-SHA256)
        │       │         │
        │       │         └─► Зашифрованные записи в vault.json
        │       │
        │       └─► Хэш верификации (domain separation: salt+"_auth")
        │
        └─ Сравнение через hmac.compare_digest() — защита от timing-атак
```

**Файл vault.json содержит только:**
- Параметры KDF (алгоритм, соль в base64)
- Верификационный хэш
- Зашифрованный Fernet-токен (все записи)

Ни одного пароля, логина или ключа в открытом виде.
